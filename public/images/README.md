# MOOSH Wallet Images

This directory contains all images and icons for the MOOSH Wallet application.

## Files

- `moosh-logo.svg` - Main MOOSH logo (vector format)
- `moosh-logo.png` - Main MOOSH logo (raster format) - needs to be added

## Logo Specifications

- Primary color: #f57315 (MOOSH orange)
- Background: #000000 (terminal black)
- Font: JetBrains Mono
- Sizes: 48x48 (standard), scalable

## Usage

In HTML:
```html
<img src="/images/moosh-logo.svg" alt="MOOSH Logo">
```

In JavaScript:
```javascript
src: '/images/moosh-logo.png'
```

If you have the original MOOSH logo files, please replace the placeholder SVG with the actual logo.
